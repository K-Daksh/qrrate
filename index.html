Welcome to Qrate 1.0

Qrate 1.0 effortlessly transforms your files into dynamic QR code videos. Written in JavaScript with Node.js as its backend technology, these videos can be conveniently uploaded to YouTube, granting you access to an abundant cloud storage solution for all your files.

*THIS IS IN DEVELOPMENT STAGE, SO PLEASE DO NOT UPLOAD FILES OVER COMBINED SIZE OF 5MBs*
